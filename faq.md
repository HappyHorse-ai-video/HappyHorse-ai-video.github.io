## How to use this site

Use the homepage if you want the big picture, the API page if your intent is developer-facing, and the official-site page if you simply want the best visible place to evaluate the product now. The goal is to shorten the path from vague search intent to a precise answer.

## Why the wording is careful

This site is SEO-driven, but it should still be trustworthy. That means avoiding invented release details, imaginary endpoints, or unsupported ownership claims simply to make a page look complete.
